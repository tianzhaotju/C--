# C--
# Project Title

使用 Yacc编写 C--(C 语言子集)语言的语法分析器，理解自上而下、自下而上的语法分析器的工作机制；掌握词法分析与语法分析之间的关系。语法分析器的输入为 C--语言源代码，输出为按扫描顺序进行分析或归约的正确/错误的判别结果。分析或归约按照语法规则进行。
